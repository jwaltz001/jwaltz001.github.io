About

Welcome to Decide & Play!
D&P was made to help you compare and decide between board games.
Simply enter the name of a board game into the search bar below, and then select the game you want from the results. Repeat for as many games as you'd like to compare.
Key statistics are displayed for each game with more information about each game also available, as well as random "choose for me" to help you decide and then... play!
